# Test App - Setup and Testing Guide 

**Prerequisites**
- Node.js - latest version
- npm 

**Setting Up the App locally**
- git clone https://github.com/alpeykov/test-app.git
- cd test-app
- npm install
- npm run server
- npm run start

**Access the test app**
App Url:
http://localhost:3000/
